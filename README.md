# clinvar_CIViC_project
SQL database project for loading and analyzing ClinVar and CIViC genetic variant data (December 2025 release).

This repository contains Python programs to process and load genetic variant data from ClinVar and CIViC (December 2025 releases) into SQL databases.

- Program A: Loads ClinVar variant information from scientific articles.
- Program B: Loads ClinVar variant statistics per gene.
- Program C: Loads variant data from CIViC, similar to ClinVar parsing.
